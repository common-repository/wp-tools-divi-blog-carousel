== Description ==

**Plugin depreciated. No further development planned for this plugin. Only maintenance and compatibility updates will be issued**

> Use our [Ultimate Carousel For Divi](https://wordpress.org/plugins/ultimate-carousel-for-divi/) plugin. It has better functionality.

The **WP Tools Divi Blog Carousel** plugin provides a divi module for cycling through blog or post items.

> Show your featured blog posts on a carousel using the **Divi Blog Carousel** module.

This plugin requires [Divi Theme](http://bit.ly/2xcicxw) (Affiliate link) be installed and activated.

**WP Tools Divi Blog Carousel** module provides following functionality.

* Full visual builder and backend builder support.
* Select unlimited number of blogs for the carousel. Order blog items with drag and drop.
* Set number of blog item per slide for desktop view. (Premium version allows you to set the items on tablet and mobile view)
* Dot navigation (Set the ability to show/hide and apply color in premium version)
* Arrow navigation (Set the ability to show/hide and apply color in premium version)

== Installation ==
1. Goto your `WordPress Admin > Plugins > Add New`
2. Search for `wp-tools-divi-blog-carousel`. 
3. Install and activate plugin (with the author as `wpt00ls`)
